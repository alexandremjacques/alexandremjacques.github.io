Vim allows you to have a project specific .vimrc file

Posted on: Sun 09 February 2020

This is interesting for some scenarios where you would prefer to keep things configured differently for a single project.

Actually, it's super simple to configure:

  1. Enable this feature by adding this to your main (located at your $HOME) .vimrc:

    set exrc

  2. Then you need to create your specific project .vimrc file configuration in the root of your project folder.

Hope it helps. :)
